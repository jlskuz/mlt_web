## Parameters

### resource

title: Device    
description:
An optional device name, number, or ID to use.  
type: string  
readonly: no  
required: no  

### audio_buffer

title: Audio buffer    
type: integer  
readonly: no  
required: no  
minimum: 256  
maximum: 8192  
default: 1024  
unit: samples  

### volume

title: Volume    
type: float  
readonly: no  
required: no  
minimum: 0.0  
default: 1.0  

### refresh

  
description:
Applications should set this to update the video frame when paused.  
type: boolean  
readonly: no  
required: no  
minimum: 0  
maximum: 1  

### scrub_audio

title: Audio scrubbing    
description:
If enabled, sound is played even when the speed is not normal.  
type: boolean  
readonly: no  
required: no  
minimum: 0  
maximum: 1  
default: 0  
widget: checkbox  

### video_delay

title: Video delay    
type: integer  
readonly: no  
required: no  
default: 0  
unit: milliseconds
