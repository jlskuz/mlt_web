## Plugin Information

title: hwmap  
media types:
Video  
description: Map hardware frames  
version: Lavfi7.16.100  
creator: libavfilter maintainers  

## Parameters

### av.mode

  
description:
Frame mapping mode  
type: string  
readonly: no  
required: no  
format: flags  
values:  

* read
* write
* overwrite
* direct

### av.derive_device

  
description:
Derive a new device of this type  
type: string  
readonly: no  
required: no  

### av.reverse

  
description:
Map in reverse (create and allocate in the sink)  
type: integer  
readonly: no  
required: no  
minimum: 0  
maximum: 1  
default: 0
